Описание:
Телеграм бот для поиска работы через сайт headhunter. C использованием API headhunter. Есть возможность проходить авторизацию 
на своем аккаунте headhunter, искать вакансии и откливаться на них, при условии что пользователь авторизован, 
также можно сохранять сопроводительное письмо для откликов.

Использованы такие технологии как express, postgres, sequelize, axios
